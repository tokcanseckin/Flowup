### ✅ Rate Limiting with Alerting (Security)
- **In-memory fixed-window rate limiter** (thread-safe, no external dependencies)
- **Rate limits applied**:
  - `POST /api/auth/login` → 5 req/15min per IP
  - `POST /api/auth/register` → 3 req/15min per IP
  - `POST /api/auth/forgot-password` → 3 req/15min per IP
  - `POST /api/auth/reset-password` → 5 req/15min per IP
  - `POST /api/auth/google` → 10 req/15min per IP
  - `POST /api/auth/apple` → 10 req/15min per IP
  - `POST /api/mailgun/webhook` → 100 req/min per IP
- **First-breach alerting system**:
  - On first rate limit violation per IP+endpoint within 24h:
    - Creates Report record (`kind='rate_limit'`) with IP, endpoint, timestamp, user agent
    - Sends email alert to personal email (not support inbox)
  - Prevents duplicate alerts within 24h window
  - Email includes security context for manual review
- **Test script**: `backend/test_rate_limit.py` for smoke testing
- **Returns 429** with `Retry-After` header when limit exceeded
